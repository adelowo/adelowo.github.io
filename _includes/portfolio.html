SCHOOLLOGUE

Schoollogue is a listing directory i built for Nigerian schools, the project aims to help guardians find the best schools for their wards - easily - regardless of the city or state. The project was borne as a result of dissatisfaction with the few school listing directories i could find.

Technologies used include : PHP (programming language), Javascript (programming language), SQL (database query language), MYSQL (database engine), HTML, CSS, Pencil for the site's mockup.
